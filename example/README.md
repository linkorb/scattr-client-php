Use these examples to test Scattr functionality

## Setup environment variables

These examples assume the following three environment variables are set:

* SCATTR_USERNAME
* SCATTR_PASSWORD
* SCATTR_ACCOUNT
* SCATTR_URL

For example:
```
export SCATTR_USERNAME="joe"
export SCATTR_PASSWORD="secret"
export SCATTR_ACCOUNT="test"
export SCATTR_URL="http://localhost:8080"
```

Now run the examples like this:
```
php ./example/add-job.php
```
